# Qf-store
Heavily heavily inspired by NERDTree.

## Feature

- qf-store-window
  - open/close qf-store
  - sort out the store
    - name
    - added at
  - Jump to the file
  - Delete item

- normal mode
  - add current qf lists to qf-store
  - add item
